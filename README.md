# 🌐 BpiRouter Package Feeds

![License](https://img.shields.io/badge/License-Proprietary/EULA-red.svg)
![Vue](https://img.shields.io/badge/Vue.js-3.x-4FC08D?logo=vue.js)
![Go](https://img.shields.io/badge/Backend-Go-00ADD8?logo=go)

Functions as a supplementary software feed for OpenWrt, hosting the package definition rules (Makefiles) for BpiRouter’s proprietary default configuration package and the next-generation web management interface (`ui`).

This feed delivers a decoupled routing architecture: a blazing-fast **Vue 3 + Naive UI** frontend SPA, powered by a highly secure **Go-based JSON-RPC API Daemon** (`bpirouter-apid`).
